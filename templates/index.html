Items Catalogue Project

Items Catalog Project

Created a catalog item application that allows users to create, read, update and delete data. This was developed using the Python framework Flask and SQL alchemy to interact with data. Some highlights of this project include:

  • Web application uses a Google sign-in to allow users to create, edit or delete items. Only users who created an item can edit or delete it
  • Personally created database. I created the database for this project myself using a database_setup.py file that creates various tables and another .py file populates these tables.

Neighbourhood Map

Neighbourhood Map

This program is an interactive map of some interesting places in Montreal that users can filter through, and get data on such as opening times and images. It uses the Bootstrap js framework to load data asynchronously, the Google Maps API for the map and various functions to interact with the map, and the Foursquare API to fetch data from the locations.
